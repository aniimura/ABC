Chapter 6: 論文I — ホッジシアターの構成

Inter-universal Teichmuller Theory I: Construction of Hodge Theaters

IUT理論の最初の論文は、理論全体の舞台となるホッジシアターを構成します。 ここでは、楕円曲線の算術データを圏論的に組織化し、 後続の論文で行われるテータリンクの構成に備えます。

初期テータデータ

IUT理論の出発点は、以下のデータの組です：

定義: 初期テータデータ

以下のデータの組 $(F/F_{\mathrm{mod}}, \overline{F}, E, \ell, \underline{\mathbb{V}}, \underline{\epsilon})$ を 初期テータデータ（initial theta data）と呼ぶ：

  1. $F$: 代数体（数体）。$F_{\mathrm{mod}} \subseteq F$ は特定の部分体
  2. $\overline{F}$: $F$ の代数閉包の選択
  3. $E$: $F$ 上の楕円曲線（半安定還元を持つ）
  4. $\ell \geq 5$: 素数（十分大きくとる）
  5. $\underline{\mathbb{V}}$: $F$ の素点の集合に関する選択
  6. $\underline{\epsilon}$: $E[\ell]$ の $\ell$ 等分点に関する選択
初学者向け補足: 初期データの直感

IUT理論を「機械」とするなら、初期テータデータはその「入力」です：

  • 楕円曲線 $E$: ABC予想を証明したい対象。ABC triple $(a,b,c)$ からフレイ曲線として構成される
  • 素数 $\ell$: 理論の「解像度」を決めるパラメータ。最終的に $\ell \to \infty$ として最良の不等式を得る
  • 素点の集合: $E$ が悪い還元を持つ素数たちの情報

出力は Szpiro不等式（= ABC予想）です。

フロベニオイド

動機: なぜ圏論的抽象化が必要か

通常、数体 $F$ の算術（イデアル、因子、直線束など）は $F$ の環構造の上で定義されます。 しかし、IUT理論ではテータリンクによって環構造が変形されます。 環構造に依存する概念は、テータリンクの先では意味を失ってしまいます。

そこで必要になるのが、環構造を直接使わず、 圏論的に算術データを記述する方法です。これがフロベニオイドの役割です。

定義: フロベニオイドの直感的定義

フロベニオイドは、以下の算術的情報を圏論的に抽象化したもの：

  • 対象: 直線束（line bundles）に対応
  • 射: 直線束の間の写像に対応
  • 次数（degree）: 射に付随する非負整数で、直線束の「大きさ」を測る
  • フロベニウス射: $d$ 倍写像に対応する特別な射（$d \geq 1$）

名前は "Frobenius" と "monoid" の合成語。

具体例: 数体のフロベニオイド

数体 $F$ に対して、モデルフロベニオイド $\mathcal{F}$ が構成されます：

  • 対象: $F$ の分数イデアルのクラス
  • 射: イデアルの包含と「$d$ 乗」操作
  • 次数: $\deg(\mathfrak{a}) = \log(\mathcal{O}_F / \mathfrak{a})$（イデアルのノルムの対数）

重要な点: フロベニオイドの圏論的構造から、 元の数体の情報（ある程度）を復元できます。つまり、情報の損失は最小限です。

遠アーベル幾何学の役割

グロタンディークの予想

1983年、グロタンディークは以下のような大胆な予想を立てました：

グロタンディークの遠アーベル予想（簡略版）

数体 $K$ 上の双曲曲線 $X$ は、そのエタール基本群 $\pi_1(X)$ によって $K$ 上の同型を除いて一意に決定される。

すなわち、$\pi_1(X) \cong \pi_1(Y)$（位相群として）ならば $X \cong Y$（$K$ 上）。

初学者向け補足: 基本群とは

基本群は、空間の「穴」の構造を記述する群です。

トポロジーでは、基本群 $\pi_1(X, x_0)$ は点 $x_0$ を始点・終点とするループの同値類の群です。 例えば、円 $S^1$ の基本群は $\mathbb{Z}$（ループの巻き数を数える）。

代数幾何では、エタール基本群が代わりに使われます。 これは、$X$ の有限エタール被覆全体を分類する副有限群です。 数体 $K$ 上では、ガロア群 $G_K$ も基本群の一種です。

遠アーベル予想は「基本群から幾何が完全に復元できる」という驚くべき主張です。

望月の単遠アーベル復元

遠アーベル予想（中島-望月、ポップらにより証明）を超えて、 望月は単遠アーベル復元（mono-anabelian reconstruction）を確立しました：

単遠アーベル復元（望月）

数体 $K$ 上の双曲曲線 $X$ に対して、$\pi_1(X)$（位相群としての構造のみ）から、 以下をアルゴリズム的に復元できる：

  • 基礎体 $K$ とその代数閉包 $\overline{K}$
  • 曲線 $X$ 自体
  • $X$ の関数体 $K(X)$
  • $K$ のすべての付値（アルキメデス・非アルキメデス）
IUTでの重要性

単遠アーベル復元は、テータリンクで環構造が壊れた後に環構造を再構成する ために使われます。基本群のデータはテータリンクを通じても保存されるため、 基本群から環構造を復元するアルゴリズムが不可欠なのです。

プライムストリップ

定義: プライムストリップ

プライムストリップは、数体 $F$ の各素点 $v$（有限素点と無限素点）での ローカルデータ（フロベニオイド、ガロア群など）を束ねた構造です。

IUT理論では複数の種類のプライムストリップが使われます：

  • $\mathcal{F}$-prime strip: フロベニオイドデータを含む（算術的情報）
  • $\mathcal{D}$-prime strip: ガロア群データを含む（群論的情報）
  • $\mathcal{D}\text{-}\Theta$-prime strip: テータ関数のデータも含む
初学者向け補足: プライムストリップの直感

数体 $F$ の各素数 $p$（と無限素点）で、楕円曲線 $E$ はそれぞれ異なる「姿」を見せます （良い還元、悪い還元など）。

プライムストリップは、これらすべての素点でのローカルな情報を 一つの「帯」（strip）としてまとめたものです。 各素点での情報が独立にパッケージ化されているため、 テータリンクで各素点の情報を個別に変形することが可能になります。

2つの対称性

ホッジシアターの構成には、$\ell$ 等分点 $E[\ell]$ に関連する2つの対称性が本質的です。

乗法的対称性: $\mathbb{F}_\ell^{\times}$

乗法的対称性

$\mathbb{F}_\ell^{\times} = (\mathbb{Z}/\ell\mathbb{Z})^{\times}$ の作用。 ラベル集合 $\{1, 2, \ldots, \lfloor \ell/2 \rfloor\}$ （テータ関数の評価点を指定するラベル）に作用します。

これはガロア群の作用と互換的で、「グローバルな」対称性を提供します。

加法的対称性: $\mathbb{F}_\ell$

加法的対称性

$\mathbb{F}_\ell = \mathbb{Z}/\ell\mathbb{Z}$ の加法群の作用。 $\ell$ 等分点全体 $E[\ell]$ に作用します。

これはテータ関数の値の「加法的な」関係を反映し、「ローカルな」対称性を提供します。

具体例: $\ell = 5$ の場合

$\ell = 5$ とすると：

  • 乗法的対称性: $\mathbb{F}_5^{\times} = \{1, 2, 3, 4\} \cong \mathbb{Z}/4\mathbb{Z}$。 ラベル集合 $\{1, 2\}$（$= \{1, \ldots, \lfloor 5/2 \rfloor\}$）に作用
  • 加法的対称性: $\mathbb{F}_5 = \{0, 1, 2, 3, 4\}$。 $E[5] \cong (\mathbb{Z}/5\mathbb{Z})^2$ に作用

乗法的対称性は「テータ関数をどの点で評価するか」の選択に関わり、 加法的対称性は「テータ関数値の和・差の関係」に関わります。

ホッジシアターの構造

これらすべての構成要素を組み合わせて、ホッジシアターが構築されます。

ホッジシアター（概略）

ホッジシアター $\mathcal{HT}^{\Theta}$ は以下の要素から構成される：

  1. $\Theta$-ホッジシアター $\mathcal{HT}^{\Theta}$: テータ関数データを持つプライムストリップの集まり。乗法的対称性 $\mathbb{F}_\ell^{\times}$ で装飾
  2. $\mathcal{D}\text{-}\Theta^{\pm\mathrm{ell}}$-ホッジシアター: $\ell$ 等分点の加法的対称性 $\mathbb{F}_\ell$ で装飾されたプライムストリップの集まり
  3. これらを結ぶ各種の同型と制約
ホッジシアターの概略構造: 乗法的対称性と加法的対称性が2つの「翼」を構成する

ラベルの役割

ラベル $j \in \{1, 2, \ldots, \lfloor \ell/2 \rfloor\}$ は、テータ関数をどの等分点で評価するかを指定します。 テータ関数 $\Theta$ の $j$ での評価値は $q^{j^2}$（Tateパラメータ $q$ の $j^2$ 乗）に関係します。

この $q^{j^2}$ という値が、後にテータリンク（Ch.7）を通じて $q$-パイロットオブジェクト（$q$ パラメータ自体）と比較されます。 $j$ が複数の値を取ることで、$q^{1}, q^{4}, q^{9}, \ldots, q^{\lfloor \ell/2 \rfloor^2}$ の 「テータパイロットオブジェクト」が得られ、これと $q$ を比較することでSzpiro不等式が導かれます。

初学者向け補足: なぜ $q^{j^2}$ が現れるか

ヤコビのテータ関数 $\theta(z, \tau) = \sum_{n=-\infty}^{\infty} q^{n^2} e^{2\pi i n z}$ （$q = e^{\pi i \tau}$）を思い出してください。 $z$ を $\ell$ 等分点 $j/\ell$ に特殊化すると、$q^{j^2}$ の項が支配的になります。

IUT理論ではこのテータ関数の $p$ 進的なバージョン（エタールテータ関数）を使いますが、 基本的な構造は同じです。

論文Iのまとめ

論文Iでは、具体的な不等式はまだ得られません。 ここでの成果は、後続の論文で使われる舞台装置の構築です：

  • 初期テータデータからホッジシアターを構成する手順の確立
  • ホッジシアターの複数のコピーを作れること
  • 乗法的・加法的対称性が正しく機能すること
  • プライムストリップ間の各種同型の確認
