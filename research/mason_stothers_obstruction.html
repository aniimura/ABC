Mason-Stothers定理の障壁分析

多項式ABCの証明はなぜ整数に持ち上がらないか

ABC予想の多項式版（Mason-Stothers定理）は1980年代に証明されました。 この証明が整数の場合に「持ち上がらない」理由を精密に分析することで、 ABC予想の本質的な困難さを理解します。

1. Mason-Stothers定理

定理 (Mason 1984, Stothers 1981)

$a(t), b(t), c(t) \in \mathbb{C}[t]$ を互いに素（共通因子を持たない）な 多項式で、$a(t) + b(t) = c(t)$ を満たすとする。このとき：

$$\max(\deg a, \deg b, \deg c) \leq n_0(abc) - 1$$

ここで $n_0(f)$ は多項式 $f$ の相異なる根の個数（= $\deg \operatorname{rad}(f)$）。

注目すべき点: 多項式版は $\epsilon$ なしで成立し、しかも $-1$ という強い形です。 整数版（ABC予想）は $1 + \epsilon$ という弱い形でさえ未証明です。

2. Mason-Stothers定理の証明

証明の鍵は微分（形式的微分 $d/dt$）の存在です。

2.1 準備：ロンスキアン

ロンスキアン（Wronskian）

2つの多項式 $f, g \in \mathbb{C}[t]$ のロンスキアンは：

$$W(f, g) = f g' - f' g$$

ここで $f' = df/dt$。$W(f, g) = 0$ は $f, g$ が比例することと同値。

補題（ロンスキアンの次数評価）

$f \in \mathbb{C}[t]$ を $0$ でない多項式とする。このとき：

$$\deg \gcd(f, f') \geq \deg f - n_0(f)$$

ここで $n_0(f)$ は $f$ の相異なる根の個数。

補題の証明

$f = c \prod_{i=1}^{k} (t - \alpha_i)^{e_i}$ と因数分解する（$e_i \geq 1$、$\alpha_i$ は相異なる根）。

$n_0(f) = k$（相異なる根の個数）。$\deg f = e_1 + \cdots + e_k$。

各根 $\alpha_i$ について、$f$ が $(t - \alpha_i)^{e_i}$ で割り切れるとき、 $f'$ は $(t - \alpha_i)^{e_i - 1}$ で割り切れる。

実際、$f = (t - \alpha_i)^{e_i} \cdot h_i(t)$ とすると、 $f' = e_i(t - \alpha_i)^{e_i - 1} h_i(t) + (t - \alpha_i)^{e_i} h_i'(t)$ $= (t - \alpha_i)^{e_i - 1}[e_i h_i(t) + (t - \alpha_i) h_i'(t)]$

よって $\gcd(f, f')$ は $\prod_i (t - \alpha_i)^{e_i - 1}$ で割り切れる。

$$\deg \gcd(f, f') \geq \sum_i (e_i - 1) = \deg f - k = \deg f - n_0(f)$$

2.2 Mason-Stothers定理の証明

証明

$a + b = c$, $\gcd(a, b) = 1$ とする。$n = \max(\deg a, \deg b, \deg c)$。

Step 1: ロンスキアンを計算する。

$a + b = c$ の両辺を微分: $a' + b' = c'$。

$W(a, c) = ac' - a'c = a(a' + b') - a'(a + b) = ab' - a'b = W(a, b)$。

同様に $W(b, c) = W(b, a) = -W(a, b)$。

$W := W(a, b) = ab' - a'b$ とおく。

Step 2: $W \neq 0$ を示す。

$W = 0$ とすると、$a/b$ は定数（$ab' = a'b$ より $(a/b)' = 0$）。 つまり $a = \lambda b$ で、$c = a + b = (\lambda + 1)b$。 $\gcd(a, b) = 1$ と $a = \lambda b$ から $\deg b = 0$、同様に $\deg a = 0$。 つまり $a, b, c$ は全て定数で、$n = 0$、$n_0(abc) \geq 0$ なので定理は自明に成立。

Step 3: $W$ の次数の上界。

$$\deg W = \deg(ab' - a'b) \leq \deg a + \deg b - 1$$

（$ab'$ と $a'b$ の次数はともに $\deg a + \deg b - 1$ 以下）

Step 4: $W$ の次数の下界。

$\gcd(a, b) = 1$ かつ $W = ab' - a'b$ より：

  • $a | W$ ではないが、$\gcd(a, a') | W$ と $\gcd(b, b') | W$ が言える：
  • $W = ab' - a'b$ において、$(t - \alpha)^{e-1}$ が $a$ と $a'$ の公約数なら、 $a' b$ は $(t-\alpha)^{e-1}$ で割れ、$a b'$ も $(t-\alpha)^e$ で割れるから $W$ も $(t-\alpha)^{e-1}$ で割れる

$\gcd(a, b) = 1$ を使うと、$\gcd(a, a')$ と $\gcd(b, b')$ は互いに素。 さらに $c = a + b$ より $\gcd(c, c')$ も $W$ を割り切る。

結果として（技術的な議論を省略すると）：

$$\deg W \geq \deg a - n_0(a) + \deg b - n_0(b)$$

実際にはより精密に、$W = ab' - a'b$ の因数構造から：

$\gcd(a, a')$, $\gcd(b, b')$, $\gcd(c, c')$ は互いに素で $W$ を割るから、

$$\deg W \geq (\deg a - n_0(a)) + (\deg b - n_0(b)) + (\deg c - n_0(c))$$

Step 5: 上界と下界を組み合わせる。

$$\deg a + \deg b - 1 \geq \deg W \geq (\deg a - n_0(a)) + (\deg b - n_0(b)) + (\deg c - n_0(c))$$

整理すると：

$$n_0(a) + n_0(b) + n_0(c) - 1 \geq \deg c$$

$n_0(a) + n_0(b) + n_0(c) = n_0(abc)$（$a, b, c$ は互いに素なので）。

$$\deg c \leq n_0(abc) - 1$$

同様に $\deg a, \deg b$ についても成立。 $\square$

3. 整数への持ち上げが失敗する箇所

上の証明で「微分」が本質的に使われています。整数環 $\mathbb{Z}$ には微分がないことが、 持ち上げの根本的な障壁です。

3.1 障壁の正確な特定

障壁1: 微分の不在

多項式環 $\mathbb{C}[t]$ には自然な微分 $d/dt$ が存在するが、 整数環 $\mathbb{Z}$ には対応する微分が存在しない。

形式的に: 環 $R$ 上の微分（derivation）$D: R \to R$ とは、 以下を満たす加法的写像：

$$D(ab) = aD(b) + D(a)b \quad (\text{ライプニッツ則})$$

$R = \mathbb{Z}$ のとき、$D(1) = D(1 \cdot 1) = 1 \cdot D(1) + D(1) \cdot 1 = 2D(1)$ より $D(1) = 0$。$D(n) = D(1 + \cdots + 1) = nD(1) = 0$。 よって $\mathbb{Z}$ 上の微分は零写像のみ。

障壁2: ロンスキアンの不在

Mason-Stothers の証明で核心的な役割を果たすのはロンスキアン $W(a, b) = ab' - a'b$ です。 微分 $'$ が存在しないので、ロンスキアンも定義できません。

ロンスキアンは「次数の上界」と「次数の下界」を同時に提供する道具であり、 これなしには上界と下界の「サンドイッチ論法」が成立しません。

障壁3: 対数微分とアルキメデス的性質

多項式の場合、$f'/f = \sum_i e_i/(t - \alpha_i)$ という対数微分の部分分数展開が 使えます。これにより $\gcd(f, f')$ と $n_0(f)$ の関係が導かれます。

整数の場合の類似物は $\log n = \sum_{p | n} v_p(n) \log p$ ですが、 これは「微分」ではなく「対数」であり、ライプニッツ則を満たしません。

4. 算術微分（Arithmetic Derivative）

$\mathbb{Z}$ 上の「微分」を人工的に定義する試みがあります。

4.1 算術微分の定義

算術微分

自然数 $n$ の算術微分 $n'$ を以下で定義する：

  • $0' = 0$, $1' = 0$
  • $p' = 1$ （$p$ が素数のとき）
  • $(mn)' = m'n + mn'$ （ライプニッツ則）

ライプニッツ則と素数の条件から一意的に定まる。

4.2 一般公式

算術微分の明示公式

$n = \prod p_i^{e_i}$ のとき：

$$n' = n \sum_i \frac{e_i}{p_i}$$

これは対数微分 $(\log n)' = \sum_i e_i / p_i$ に対応する （$n' = n \cdot (\log n)'$ の形式的類似）。

4.3 算術微分の限界

なぜ算術微分ではABC予想を証明できないか
  1. 加法との非整合性: $(a + b)' \neq a' + b'$ が一般的。例: $(2 + 3)' = 5' = 1$ だが $2' + 3' = 1 + 1 = 2$。算術微分は加法に関する微分ではない
  2. ロンスキアンの構成不能: $a + b = c$ から $a' + b' = c'$ が導けないので、 ロンスキアン的な議論が成立しない
  3. 次数の概念の不在: 多項式の次数 $\deg$ に相当する「大きさ」は $\log |n|$ だが、 $\log |n'|$ と $\log |n|$ の間に $\deg f'$ と $\deg f$ の間のような 単純な関係がない
  4. 体の構造の欠如: $\mathbb{C}[t]$ は体 $\mathbb{C}(t)$ に埋め込めるが、 $\mathbb{Z}$ は $\mathbb{Q}$ に埋め込んでも算術微分は 加法と整合しない

5. ロンスキアンの算術的類似物の探索

5.1 望まれる性質

理想的な「算術的ロンスキアン」

整数 $a, b$ に対して、以下の性質を持つ量 $W(a, b)$ が存在すれば、 Mason-Stothersの議論が持ち上がる可能性がある：

  1. $W(a, b) = 0 \Leftrightarrow a/b$ が「定数」的
  2. $\log |W(a, b)| \leq \log |a| + \log |b| - 1$ （次数の上界に対応）
  3. $\log |W(a, b)| \geq (\log |a| - \log \operatorname{rad}(a)) + (\log |b| - \log \operatorname{rad}(b))$ （次数の下界に対応）

このような $W$ が見つかれば、ABC予想が証明される。

5.2 既知の試み

候補1: 行列式的構成

$a, b$ の素因数分解 $a = \prod p_i^{a_i}$, $b = \prod p_i^{b_i}$ に対して、

$$W(a, b) = \det \begin{pmatrix} a_1 & a_2 & \cdots \\ b_1 & b_2 & \cdots \end{pmatrix}$$

の一般化。しかし、これは有限個の素数にしか対応せず、 無限個の素数を扱う方法が確立されていない。

候補2: アラケロフ理論的高さ

アラケロフ理論では、数体上の多様体に対する「高さ」関数が定義され、 これは関数体の場合の次数に対応する。

しかし、アラケロフ的高さは個々の整数に対する「微分」を直接与えない。 IUT理論はこの障壁を、ホッジ・アラケロフ理論の一般化として乗り越えようとしている。

候補3: p進微分

各素数 $p$ に対して、$p$ 進体 $\mathbb{Q}_p$ 上では $p$ 進微分（$p$ 進対数など）が 存在する。これを全ての素数 $p$ について統合する方法として：

  • アデール的構成: $\mathbb{A}_{\mathbb{Q}} = \prod'_p \mathbb{Q}_p$ 上の微分?
  • しかし、アデール上の微分と整数環上の微分は本質的に異なる

IUT理論の対数リンクは、$p$ 進対数 $\log_p$ を用いて 「$p$ 進微分的」な操作を実現しようとするものと解釈できる。

6. 圏論的分析: なぜ「微分」が存在しないか

6.1 算術幾何の基本的な非対称性

数体と関数体の根本的差異

関数体 $\mathbb{C}(t)$ と数体 $\mathbb{Q}$ の間には深い類似がありますが、 決定的な差異も存在します：

性質$\mathbb{C}(t)$$\mathbb{Q}$

幾何的構造$\mathbb{P}^1_{\mathbb{C}}$ 上の関数$\operatorname{Spec} \mathbb{Z}$ 上の「関数」
基礎体$\mathbb{C}$（代数閉体）$\mathbb{F}_1$?（「1元体」）

6.2 ケーラー微分の消滅

$\Omega_{\mathbb{Z}/\mathbb{Z}} = 0$ の証明

$\mathbb{Z}$ の $\mathbb{Z}$ 上のケーラー微分加群は零：

$$\Omega_{\mathbb{Z}/\mathbb{Z}} = 0$$

証明: ケーラー微分は $d(ab) = a \cdot db + b \cdot da$ と $da = 0$ (for $a \in \mathbb{Z}$, base ring) を満たす。$\mathbb{Z}$ over $\mathbb{Z}$ では全ての元が基底環に属するので $dn = 0$ for all $n$。

これは「$\mathbb{Z}$ は $\mathbb{Z}$ 自身の上ではエタールである」ことの反映であり、 微分幾何的な「自由度」が存在しないことを意味します。

6.3 F_1 上の幾何学?

「1元体」$\mathbb{F}_1$ のアイデア

Mason-Stothersの証明が機能するのは、$\mathbb{C}[t]$ が $\mathbb{C}$ 上の多項式環であり、 「$\mathbb{C}$ に対する相対的な微分」$d/dt$ が存在するからです。

$\mathbb{Z}$ をある「基礎対象」$\mathbb{F}_1$（仮想的な1元体）上の 「多項式環」$\mathbb{F}_1[t]$ と見なすことができれば、 $\mathbb{F}_1$ に対する相対的な微分が定義できる可能性があります。

$\mathbb{F}_1$ 上の代数幾何学は活発な研究分野であり（Connes-Consani, Borger, Durov等）、 ABC予想の証明戦略の一つとして注目されています。

しかし現時点では、$\mathbb{F}_1$ の定式化は複数の互いに非同値な提案があり、 Mason-Stothersの証明を持ち上げるのに十分な理論は確立されていません。

6.4 まとめ：障壁の本質

障壁の圏論的まとめ

Mason-Stothersの証明が整数に持ち上がらない根本的な理由は：

  1. $\mathbb{Z}$ は「絶対的な」対象であり、 相対的な微分（ケーラー微分）が消滅する
  2. 多項式環の微分が与える「次数の制御」に相当する道具が、 整数環には自然には存在しない
  3. この障壁を乗り越えるための主なアプローチ:
    • IUT理論: テータリンクと対数リンクの組み合わせで、 「微分」の代替を提供しようとする
    • $\mathbb{F}_1$ 幾何学: 「基礎体」の概念を拡張して 相対微分を回復しようとする
    • アラケロフ幾何学: 高さ関数を通じて 「次数」の類似物を提供する
    • 楕円曲線/モジュラー形式: ABC予想をSzpiro予想に翻訳し、楕円曲線の理論で攻める
