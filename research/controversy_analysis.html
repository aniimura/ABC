Scholze-Stix vs 望月 論争の精密数学的分析

IUT理論の中核的論争点の厳密な検討

1. 論争の概要

2018年、Peter Scholze と Jakob Stix は望月新一のIUT理論（宇宙際タイヒミューラー理論）に対する 根本的な批判を公表しました。この論争は2025年現在も決着していません。

2. Scholze-Stix の指摘の正確な内容

2.1 論点の核心

Scholze-Stixの主張は、一言で言えば次のとおりです：

Scholze-Stix の主張（概要）

IUT理論の証明において、テータリンクが与える対応

$$q^{j^2} \longleftrightarrow q$$

を正当化するために使われる「異なる宇宙」の区別は不要であり、 全てを1つの宇宙で考えると、テータリンクは自明な恒等写像に帰着し、 非自明な不等式は得られない。

より具体的には、Scholze-Stixの批判は以下の3つのレベルで展開されます：

2.2 第1の論点：ホッジシアターの「単純化」

Scholze-Stix の「単純化」の手順

Scholze-Stixは次のように論じました：

  1. IUT理論のホッジシアター $\mathcal{HT}$ は、数体 $F$ のデータ（素点での局所データの集合）から構成される
  2. 「異なるホッジシアター」${}^{n}\mathcal{HT}$ と ${}^{n+1}\mathcal{HT}$ は、同じ数体 $F$ から構成されているのだから、それらは同型である
  3. テータリンクは「2つの同型なコピー」の間の写像に過ぎず、共通の $F$ の上で考えれば自明になる
  4. 結果として、テータリンクから非自明な不等式を引き出すことは不可能

2.3 第2の論点：系3.12の証明

系3.12への疑問

IUT論文IIIの系3.12（Corollary 3.12）は、理論の最重要な結果です：

$$-|\log(\Theta)| \leq -|\log(q)| + C$$

ここで $|\log(\cdot)|$ は対数体積を表す。

Scholze-Stixは、この系の証明において：

  • テータパイロットオブジェクト $\{q^{j^2}\}_{j=1}^{\lfloor\ell/2\rfloor}$ と $q$-パイロットオブジェクト $\{q\}$ の区別が、 「単純化」によって消えてしまう
  • したがって不等式は $-|\log(q)| \leq -|\log(q)| + C$ という自明な主張に帰着する

2.4 第3の論点：不定性（Indeterminacies）の役割

IUT理論では3つの不定性 (Ind1), (Ind2), (Ind3) が本質的な役割を果たしますが、 Scholze-Stixはこれらについても疑問を呈しました：

不定性に関する Scholze-Stix の見解
  • (Ind1): エタール基本群の内部自己同型による不定性は、 通常の圏論で標準的に処理できるはず
  • (Ind2): $\widehat{\mathbb{Z}}^{\times}$ の作用による不定性は、 テータリンクの構成が well-defined であるための技術的条件に過ぎない
  • (Ind3): 対数リンクによる加法的不定性は、 「単純化」の下では自明になる

要するに、Scholze-Stixの立場では、これらの不定性は人工的に導入された複雑さであり、 証明に本質的な寄与をしていないと主張します。

3. 「単純化」の数学的詳細

3.1 Scholze-Stix の論法の再構成

Scholze-Stixの核心的な議論を、可能な限り厳密に再構成します。

設定

$F$ を数体、$E/F$ を楕円曲線（分裂悪還元型）、 $\ell$ を十分大きい素数とする。IUT理論の初期テータデータとする。

IUT理論は、同じ初期データから「異なるコピー」${}^{n}\mathcal{HT}$ を構成し、 これらをテータリンクで接続する。

Scholze-Stix の「定理」（彼らの主張）

主張: IUT理論の構成を「1つの宇宙」の中で考えると、 テータリンクは以下のように簡約される：

  1. $\mathcal{O}_{\bar{F}_v}^{\triangleright}$（$v$ が悪還元素点）に対して、 テータパイロットオブジェクトは $q_v^{j^2} \in \mathcal{O}_{\bar{F}_v}^{\times}$ （ここで $q_v$ は Tate パラメータ）
  2. テータリンクは $q_v^{j^2} \mapsto q_v$ という写像を与える
  3. しかし $q_v^{j^2}$ と $q_v$ は同じ $\mathcal{O}_{\bar{F}_v}$ の元であり、 この写像は単に「$j^2$ 乗する」操作の逆に過ぎない

結果として、テータリンクを通じて得られる不等式は自明。

3.2 「単純化」が意味すること

Scholze-Stixの「単純化」とは、技術的には次のことを意味します：

圏論的な視点での「単純化」

IUT理論では、ホッジシアターは以下のデータを含む：

  • 「基礎体」 $F$ のコピー（ただし異なるラベル ${}^{n}F$）
  • フロベニオイド（環構造のエンコード）
  • エタール基本群の作用

Scholze-Stixの単純化: ${}^{n}F = {}^{m}F = F$（ラベルを外す）と見なすと、 全ての構成は $F$ の上の通常の代数幾何学に帰着する。

テータリンク $\Theta: {}^{n}\mathcal{HT} \to {}^{n+1}\mathcal{HT}$ は、 $F$ 上のフロベニオイド間の写像 $q^{j^2} \mapsto q$ に「退化」する。

3.3 具体例での図示

具体例: $\ell = 5$ の場合

$\ell = 5$ とすると、$j = 1, 2$（$j = 1, \ldots, \lfloor 5/2 \rfloor$）。

テータパイロットオブジェクトは $\{q^1, q^4\}$。

IUT の見方：

  • ${}^{0}\mathcal{HT}$ の世界: テータパイロット $\{q^1, q^4\}$ が住む
  • ${}^{1}\mathcal{HT}$ の世界: $q$-パイロット $\{q\}$ が住む
  • テータリンク: $\{q^1, q^4\} \xleftrightarrow{} \{q\}$
  • 2つの世界は異なる「宇宙」であり、環構造は共有されない
  • この非共有性から非自明な不等式が生じる

Scholze-Stix の見方：

  • $q^1, q^4, q$ は全て同じ $\mathcal{O}_{\bar{F}_v}$ の元
  • 「異なる宇宙」は数学的に正当化されない
  • $q^1 \leftrightarrow q$ は自明（$q^1 = q$）
  • $q^4 \leftrightarrow q$ は「$q$ を $1/4$ 乗する」操作で自明
  • 不等式の導出は循環論法

4. 望月の反論

4.1 反論の骨子

望月の反論（核心）

望月の反論は、Scholze-Stixの「単純化」が理論の本質を破壊していると主張します：

  1. 「宇宙際」思考 vs 「宇宙内」思考: Scholze-Stixは全てを1つの宇宙 $\mathcal{V}$ の中で考えているが、 IUT理論の核心は複数の宇宙 $\mathcal{V}_0, \mathcal{V}_1, \ldots$ を 同時に扱うことにある
  2. 環構造の非共有: 異なるホッジシアター間では環構造（加法と乗法の同時的な関係）が 共有されないことが本質的。これを無視する「単純化」は、 証明の論理構造を破壊する
  3. 多輻的アルゴリズム: IUT理論の結論は「多輻的」（multiradial）な方法、すなわち テータリンクの不定性に対して不変な方法で得られる。 この構造はScholze-Stixの単純化では見えなくなる

4.2 「宇宙際」思考の数学的内容

望月の「宇宙際」概念

IUT理論では、各ホッジシアター ${}^{n}\mathcal{HT}$ は 独自の「宇宙」$\mathcal{V}_n$ に住むとされます。

形式的には：

  • $\mathcal{V}_n$ は $\mathrm{ZFC}$ 集合論のモデル
  • ${}^{n}\mathcal{HT}$ は $\mathcal{V}_n$ の中の対象
  • テータリンクは $\mathcal{V}_n$ から $\mathcal{V}_{n+1}$ への「部分的な」対応

望月の主張: $\mathcal{V}_n$ と $\mathcal{V}_{n+1}$ を「同じ宇宙」と見なすことは、 テータリンクが環構造を変形するという本質的な操作を消去してしまう。

直感的な理解の試み

2つの見方の違いを、日常的な例で考えてみましょう：

座標変換のアナロジー:

  • 2次元平面で、直交座標 $(x, y)$ と極座標 $(r, \theta)$ を考える
  • 「全て同じ平面の点だから、座標系の違いは本質的でない」（Scholze-Stix的）
  • 「いや、座標変換の過程で情報が変形される。その変形量が重要」（望月的）

IUTの場合、テータリンクは座標変換よりも遥かに激しい「変形」です。 環構造（加法と乗法）を同時に保つことができない変換であり、 その「歪み」の量が最終的な不等式を与えます。

4.3 望月の具体的な反論点

望月の反論の数学的ポイント

反論1: ラベルの同一視は非合法

${}^{n}F$ と ${}^{m}F$ を同一視すると、テータリンクの構成で使われる フロベニオイドの乗法的構造と加法的構造の間の「ギャップ」が消えてしまう。 このギャップこそが不等式の源泉である。

反論2: 「自明化」は証明の破壊

Scholze-Stixの操作は、テータリンクだけでなく、それを取り巻く 対数リンク・多輻的表現・不定性の体系全体を破壊する。 これは「単純化」ではなく「破壊」である。

反論3: 圏論的な厳密性

IUT理論は「圏の圏」（2-圏）のレベルで構成されており、 対象のレベルでの同型が、射のレベルでの非自明性を消去するとは限らない。 Scholze-Stixは対象のレベルの同型のみを議論している。

5. 系3.12のギャップの数学的定式化

5.1 系3.12の内容

系3.12 (IUT III, Corollary 3.12)

適切な初期テータデータの下で、以下の不等式が成り立つ：

$$-|\log(\Theta)| \leq -|\log(q)| + \text{(定数項)}$$

ここで：

  • $|\log(\Theta)|$ はテータパイロットオブジェクト $\{q^{j^2}\}$ の対数体積
  • $|\log(q)|$ は $q$-パイロットオブジェクト $\{q\}$ の対数体積
  • 定数項はデータ $(F, E, \ell)$ に依存

この不等式から、適切な $\ell$ の選択により Szpiro 不等式が導かれる。

5.2 Scholze-Stix が指摘するギャップ

ギャップの所在

系3.12の証明は、大まかに以下のステップで構成されます：

  1. テータリンクにより、$\{q^{j^2}\}$ と $\{q\}$ を「比較可能」にする
  2. 多輻的表現により、テータリンクの不定性 (Ind1, Ind2, Ind3) を「吸収」する
  3. 対数殻（log-shell）$\mathcal{I}_v$ を共通の「容器」として使い、 両者の対数体積を同じ空間で測る
  4. テータパイロットが対数殻に「収まる」ことから上界を得る

Scholze-Stix が疑問視するのはステップ1： テータリンクによる「比較」が、1つの宇宙では自明（$q^{j^2}$ と $q$ は同じ環の元） になるため、ステップ3-4で得られる不等式も自明になるという主張です。

5.3 問題の数学的本質

問題の形式化

問題を形式化すると、以下のようになります：

IUT の主張: テータリンクは「モノイド構造のみ」（乗法のみ）を保つ写像であり、 加法的構造は保たない。従って：

$$\Theta: (\mathcal{O}_v^{\triangleright}, \times) \to (\mathcal{O}_v^{\triangleright}, \times)$$

は well-defined だが、

$$\Theta: (\mathcal{O}_v, +, \times) \to (\mathcal{O}_v, +, \times)$$

は定義されない（環の射ではない）。

この「加法的構造の欠落」により、テータパイロットの対数体積が $q$-パイロットの対数体積と異なることが可能になる。

Scholze-Stix の反論: しかし、$\mathcal{O}_v$ の環構造は「客観的に存在する」のであり、 テータリンクが環構造を「忘れる」からといって、 $q^{j^2}$ と $q$ の間の（既知の）環的関係が消えるわけではない。

アナロジー: 「忘却関手」の問題

この問題は、忘却関手に関する以下の哲学的問題に類似しています：

  • $\mathbb{R}$ を位相空間と見なす（群構造を忘れる）
  • $\mathbb{R}$ を群と見なす（位相構造を忘れる）
  • これらの「忘却」は正当な数学的操作である

IUTでは：

  • $\mathcal{O}_v$ の環構造を「忘れて」モノイド構造のみを残す
  • この忘却の下で $q^{j^2} \mapsto q$ の写像を構成する
  • 忘却された加法構造の「不在」から不等式が生じる

Scholze-Stixの反論: 忘却関手を適用しても、元の環構造の情報は消えない。 関手の像だけを見て結論を出すのは不正当。

6. Joshi のアプローチ

6.1 Joshi の算術的タイヒミューラー空間

Joshi の構成 (2023-2024)

Kirti Joshi（アリゾナ大学）は、IUT理論の代替的な定式化を提案しました：

  • パーフェクトイド体（perfectoid fields）の理論を用いて、 望月のフロベニオイドに相当する構造を再構成
  • Scholze自身が開発したパーフェクトイド理論の言語で IUT的な構成を記述することで、批判に答えようとする
  • 「算術的タイヒミューラー空間」$\mathcal{AT}_p$ を定義し、 IUT理論の核心的構成を異なる角度から正当化

6.2 Joshi のアプローチの要点

Joshi の主要な主張
  1. フロベニウス持ち上げの非一意性: $p$ 進体 $K$ の「フロベニウスの持ち上げ」は一般に一意ではなく、 その非一意性がタイヒミューラー空間的な「変形空間」を形成する。 これがIUT理論で「異なるホッジシアター」に対応する
  2. パーフェクトイド化: $K$ のパーフェクトイド化 $K^{\flat}$ と $K$ の間の tilting 対応は、 環構造を保たない（加法は保存されない）。 これはテータリンクの非環構造的性質と対応する
  3. Scholze の理論との接続: パーフェクトイド理論は Scholze 自身が2012年に開発したものであり、 その言語でIUTを記述することは、Scholze-Stixの批判に 直接応答する意味がある

6.3 望月によるJoshi批判

望月の反応 (2024)

望月は2024年にJoshiの論文に対する詳細な批判レポートを公開しました：

  • Joshiの構成は IUT の本質的な部分を捉えていない
  • 特に「多輻的アルゴリズム」の概念が正しく反映されていない
  • Joshiの「算術的タイヒミューラー空間」は IUT のホッジシアターとは本質的に異なる

これにより、「第三者によるIUTの独立な検証」という期待も複雑化しています。

7.2 解決に向けた可能性

想定される解決シナリオ
  1. 形式化による決着: Lean4 等の証明支援系でIUT理論を形式化し、系3.12の証明を機械検証する。 これが最も客観的な決着方法だが、理論の複雑さから数十年かかる可能性がある
  2. 独立な証明の発見: IUTとは異なる方法でABC予想を証明する。 現状の最有力候補は Pasten (2024) の志村曲線アプローチ
  3. Scholze-Stix の反例構成: 「単純化」の下で反例（ABC予想に反するケース）を構成する。 ただし、ABC予想自体は正しいと広く信じられているため、 この方向は「IUTの証明が間違い」を示すだけで反例にはならない
  4. 概念的な橋渡し: Joshi型のアプローチにより、IUTの核心的構造を Scholze等が受け入れ可能な形で再定式化する

7.3 数学的教訓

この論争から学べること
  • 証明の検証可能性: 数学は「正しさ」が客観的に判定できると信じられていますが、 十分に複雑な理論では、専門家の間でも合意が得られないことがあります。 形式化・機械検証の重要性を示す事例です
  • 新しい概念の受容: 「宇宙際」の概念は既存の数学の枠組みを超えようとするものであり、 その正当性の判断自体が困難です。 ガロアの群論やカントールの集合論も、当初は受け入れに時間がかかりました
  • 数学のコミュニケーション: 700ページを超える論文の理解・検証は、個人の能力を超える課題であり、 数学コミュニティ全体での取り組みが必要です
