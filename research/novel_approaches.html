IUT以外の最新研究動向と可能性の分析

2. Pasten (2024): 志村曲線アプローチ

2.1 背景: Stewart-Yuの結果

Stewart-Yu (2001) の定理

$a + b = c$, $\gcd(a, b) = 1$ のとき：

$$\log \max(|a|, |b|, |c|) \ll \operatorname{rad}(abc)^{1/3} (\log \operatorname{rad}(abc))^3$$

ここで暗黙の定数は非常に大きい（$\sim 1.7 \times 10^{30}$）。

ABC予想は $\operatorname{rad}(abc)^{1+\epsilon}$ を主張しているのに対し、 最善の無条件結果は $\operatorname{rad}(abc)^{1/3}$ のオーダーに留まっています。

この $1/3$ の指数は20年以上改善されていませんでした。

2.2 Pasten の突破口

Pasten (2024) の結果

Hector Pasten（チリ・カトリカ大学）は、志村曲線（Shimura curves）を用いて Stewart-Yuの結果を初めて改善しました。

主要結果（概略）：ある明示的な $\delta > 0$ が存在して、

$$\log c \ll \operatorname{rad}(abc)^{1/3 + \delta}$$

（正確な値は $\delta$ の下界とともに明示的に与えられる）

2.3 Pasten の方法の概略

志村曲線アプローチの仕組み

従来のABC予想へのアプローチは、フレイ曲線 $E_{a,b,c}: y^2 = x(x-a)(x+b)$ を 構成し、そのモジュラー性を利用するものでした（ワイルズのFLT証明と同じ戦略）。

Pastenは以下の革新を導入：

  1. 志村曲線の利用: モジュラー曲線 $X_0(N)$ の代わりに、不定四元数環から構成される 志村曲線 $X^D$ を使用
  2. フレイ-Hellegouarch曲線の一般化: フレイ曲線をより一般的な族に拡張し、志村曲線へのモジュラーパラメトリゼーションを 活用
  3. 高さの制御: 志村曲線の算術的高さ理論を精密に解析し、Stewart-Yuの $1/3$ 指数を超える
なぜ志村曲線が有効か

志村曲線はモジュラー曲線の一般化であり、より豊かな算術的構造を持ちます。

  • モジュラー曲線は「1次元ゼロの超構造」（行列代数 $M_2(\mathbb{Q})$）から構成
  • 志村曲線は「非自明な四元数環 $D$」から構成され、追加の対称性を持つ
  • この追加の対称性が、高さの評価に新しい道具を提供する

歴史的に、Ribet のレベル下げ定理は志村曲線でも機能するため、 FLT型の議論を志村曲線上で展開できます。

2.4 発展の可能性

Pasten の方法の限界と可能性
  • 現時点では $1/3 + \delta$ の $\delta$ は非常に小さく、 $1 + \epsilon$ には程遠い
  • しかし、20年間動かなかった壁を超えた意義は大きい
  • 志村曲線の族を系統的に活用することで、$\delta$ を大きくできる可能性がある
  • 異なるレベルの四元数環を使い分けることで、 各素数 $p$ に対する局所的な制御を改善できるかもしれない

3. Browning-Lichtman-Teravainen (2024-2025): 統計的ABC

3.1 結果の概要

「ABCはほとんど全ての場合に成立する」

Tim Browning, Jared Lichtman, Joni Teravainen は2024年に以下を示しました：

任意の $\epsilon > 0$ に対し、$c \leq X$ の範囲で $c > \operatorname{rad}(abc)^{1+\epsilon}$ を満たす ABC triple $(a, b, c)$ の数は、

$$\ll X^{1 - \delta(\epsilon)}$$

ここで $\delta(\epsilon) > 0$ は $\epsilon$ に依存する明示的定数。 つまり、ABC予想に「反する」triple の数は、全体の triple 数よりも 冪乗的に小さい（power saving）。

これは何を意味するか

ABC予想は「有限個」の例外しかないと主張しますが、 Browning et al. は「ほとんどない」ことを証明しました：

  • $c \leq X$ の ABC triple は全部で $\sim X$ 個のオーダー
  • そのうち ABC 予想に反するものは $\leq X^{1-\delta}$ 個
  • 割合 $X^{1-\delta}/X = X^{-\delta} \to 0$ ($X \to \infty$)

「有限個」の証明ではないが、「密度0」よりも遥かに強い結果です。

3.2 方法の概略

解析的数論の道具

Browning et al. の方法は、IUTや代数幾何とは全く異なる、 解析的数論の道具を用います：

  1. 篩法（Sieve methods）: セルバーグの篩、大きな篩を用いて、 小さい rad を持つ整数の分布を制御
  2. 指数和の評価: $\sum_{n \leq X} e^{2\pi i f(n)/q}$ 型の和の評価（Weyl, Vinogradov）
  3. $L$ 関数の零点分布: ディリクレ $L$ 関数の零点の分布に関する結果を利用
  4. 2重篩: $a$ と $c - a$ の両方に篩的条件を課す技法

3.3 このアプローチの限界

なぜ完全な証明には至らないか
  • 篩法は「密度」的な結果（大部分の整数について成立）を得るのに適しているが、 「全ての」整数について示すことは原理的にできない
  • 爆弾的な反例（万が一存在する場合）は篩法では検出できない
  • 「有限個」の証明にはまったく異なる手法が必要
  • ただし、ABC予想が正しいという強い数値的証拠を提供する

4. Zhou Zhongpeng (2025): 有効的ABC不等式

4.1 対数形式の理論

Baker の対数形式の定理

Alan Baker (Fields Medal 1970) は以下の基本定理を証明しました：

代数的数 $\alpha_1, \ldots, \alpha_n$ と整数 $b_1, \ldots, b_n$ に対して、

$$\Lambda = b_1 \log \alpha_1 + \cdots + b_n \log \alpha_n \neq 0$$

ならば $|\Lambda| > B^{-C}$（$B = \max |b_i|$、$C$ は $\alpha_i$ に依存する有効定数）。

この「対数形式の下界」は、ディオファントス方程式の有効的解法の基礎です。

4.2 Zhou の改善

Zhou (2025) の有効的ABC不等式

Zhouは対数形式の理論を精密に適用して、以下を示しました：

$$\log |abc| \leq 3 \log \operatorname{rad}(abc) + 8\sqrt{\log |abc| \cdot \log \log |abc|}$$

この不等式は有効的（全ての定数が明示的）であり、 従来の Stewart-Yu の定数（$K \sim 1.7 \times 10^{30}$）を $K = 400$ に改善しました。

Zhou の結果の意義
  • 定数の劇的改善: $10^{30}$ から $400$ への改善は 実用的な意味で大きい
  • 計算的検証: 小さな定数により、コンピュータで検証可能な範囲が拡大
  • 指数の壁: しかし指数は依然として $1/3$ のオーダー ($3 \log \operatorname{rad}$)であり、ABC予想の $1 + \epsilon$ には程遠い
  • 方法論的限界: Baker理論に基づく限り、指数 $1/3$ の壁は 本質的である可能性が高い

4.3 Baker 理論の限界

指数 1/3 の壁

Baker の対数形式の理論から ABC予想に向かうとき、 以下の壁に直面します：

  1. ABC triple $a + b = c$ を $\log(c/a) = \log(1 + b/a)$ の形に書く
  2. Baker の定理から $|\log(c/a)|$ の下界を得る
  3. この下界は $\operatorname{rad}(abc)$ の冪の逆数の形をしている
  4. しかし、Baker理論の最良の結果でも指数は $-1/3$ 程度 （$|\Lambda| \geq \operatorname{rad}^{-1/3+\epsilon}$ の形）

ABC予想が要求する $c < \operatorname{rad}(abc)^{1+\epsilon}$ を得るには、 指数が $-1$ を超える必要がありますが、 現在の対数形式の理論では $-1/3$ が限界です。

5. 計算複雑性との接続

5.1 ABC予想と計算量

CCC 2024の結果

Computational Complexity Conference 2024 において、 ABC予想と計算量理論の接続に関する結果が発表されました。

  • ABC予想が成立するならば、特定の計算問題に対する 下界が得られる
  • 逆に、ある種の計算量的仮定からABC予想の弱い版が導かれる
rad(n) の計算複雑性

$\operatorname{rad}(n)$ の計算は素因数分解と等価であり、 これは計算量理論の中心的な問題です：

  • $\operatorname{rad}(n)$ の計算: 素因数分解が必要 → NP中間と予想される
  • 量子コンピュータではShorのアルゴリズムで多項式時間
  • ABC予想は「$\operatorname{rad}(abc)$ が大きい」ことを主張しており、 これは「素因数分解が容易でない数は少ない」ことと関連

5.2 ABC予想と暗号理論

暗号理論への含意

ABC予想が成り立つならば：

  • 大きな素因数を持つ整数の分布に関する定量的結果が得られる
  • 楕円曲線暗号（ECC）の安全性に関する数論的基礎が強化される
  • 特に、楕円曲線の判別式と導手の関係（Szpiro不等式）は、 ECCの効率的な実装と安全性評価に直結する

6. 力学系アプローチ

6.1 算術力学とABC

算術力学の基本

算術力学（arithmetic dynamics）は、力学系の理論を 数論に適用する分野です。

基本設定: 有理写像 $\phi: \mathbb{P}^1 \to \mathbb{P}^1$ のイテレーション $\phi, \phi^2 = \phi \circ \phi, \phi^3, \ldots$ を考える。

$\phi(x) = x^2 + c$ のような写像に対して：

  • 周期軌道 $\phi^n(x) = x$ の算術的性質
  • 前周期点（preperiodic points）の高さの評価
  • 正準高さ $\hat{h}_\phi$ とネロン-テイト高さの類似
力学系版ABC予想

Silverman (1993, 2007) は力学系的な ABC 予想の類似物を定式化しました：

$\phi(x) = x^d + c$ のイテレーション $\phi^n(a) = b$ に対して、 $\operatorname{rad}$ 的な量が $b$ の「大きさ」を制御するか？

  • $d = 1$（線形写像）の場合、通常のABC予想に帰着
  • $d \geq 2$ の場合、力学系的な構造が追加の制約を与える
  • 特に $d = 2$ の場合、マンデルブロ集合との接続がある

力学系的な視点はABC予想に新しい洞察を与える可能性がありますが、 現時点では直接的な証明への道筋は見えていません。

6.2 p進力学系とIUT

p進力学系の観点

IUT理論のフロベニウス持ち上げは、$p$ 進力学系の文脈で理解できます：

  • フロベニウス写像 $\sigma: x \mapsto x^p$ は $\mathbb{F}_p$ 上の力学系
  • その $\mathbb{Z}_p$ への「持ち上げ」$\tilde{\sigma}$ は一意でない
  • $\tilde{\sigma}$ の選択の空間がJoshiの「算術的タイヒミューラー空間」
  • IUT理論はこの空間の「変形」を追跡し、テータリンクで接続する

この観点は IUT と力学系を橋渡しする可能性を示唆しています。

7.2 最も有望な方向

個人的評価（2025年2月時点）
  1. IUT論争の解決（最高優先度）: IUT理論が正しいかどうかの決着が、ABC予想の歴史を決定する。 Lean4等による形式化が最も客観的な解決策だが、 実現には長期間を要する
  2. Pastenの方法の発展（中期的に有望）: 志村曲線アプローチは、20年ぶりの壁突破を達成した。 指数を $1/3$ から $1$ に近づけることができるかが鍵。 異なる志村曲線の族を系統的に使う方法の探索が重要
  3. 新しい「微分」の発見（革新的だが困難）: Mason-Stothers障壁の分析が示すように、 整数環上の「微分」に相当する道具の発見がブレークスルーの鍵。 $\mathbb{F}_1$ 幾何学、IUTの対数リンク、パーフェクトイド理論のいずれかが この道具を提供する可能性がある
  4. 意外な接続の発見（長期的な夢）: ラングランズ・プログラム、モチーフ理論、 量子場理論との接続など、 予想外の分野からの着想がブレークスルーを生む可能性

7.3 計算実験から見える手がかり

計算的観察

本プロジェクトの計算実験（abc_large_search.py） から得られた観察：

  1. 最高quality tripleの構造: 高いquality を持つ ABC triple は、$a$ または $b$ が 「滑らかな数」（smooth number: 小さい素因数のみ）である傾向がある
  2. quality分布: quality > 1 の triple は非常に稀であり、その割合は $c$ の増加とともに減少する（ABC予想の数値的証拠）
  3. Szpiro比との相関: quality と Szpiro比は強い正の相関を持ち、 ABC予想とSzpiro予想の等価性を数値的に確認
  4. Zhou不等式の余裕: 全ての検証例でZhouの有効的ABC不等式は大きな余裕を持って成立

$100万ドルへの道
